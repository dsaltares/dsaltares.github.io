# saltares.com blog

Personal blog built using Next.js.

## Requirements

- Node 16, fnm recommended
- yarn

## Run locally

Copy the `.env.sample` file and edit according to your needs.

```bash
cp .env.sample .env
```

Then:

```bash
yarn install
yarn dev
```

## Deployment

It happens automatically every time code is pushed to `main` using Github Actions.
