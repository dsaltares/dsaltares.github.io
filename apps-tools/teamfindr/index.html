Team Findr is open source under the MIT license. Contributions are welcome! The application is entirely written in Typescript and it has two main components.

  • web-app: client side React application. It uses Service Workers, React Query, MaterialUI, Leaflet and Formik.
  • api: REST API, deployed to an EC2 machine. It uses express, mongodb and Passport.
