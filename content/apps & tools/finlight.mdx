💸 [Finlight](https://github.com/dsaltares/finlight-electron) is a desktop personal finance app. It's free and open source.

![finlight logo](/img/finlight/finlight.png)

I made it because I couldn't find a personal finance app that met my needs.

### 🚀 Features

- 🔐 Local and privacy first, you 100% own your data.
- 🏦 Multi-account.
- 💱 Multi-currency support with currency normalisation.
- 💸 Track income, expenses and account transfers (double-entry).
- 📊 Insights.
- 💰 Budgeting.
- 🙌 Cross platform: Linux, Windows and Mac.

### 💻 Technical details

[Finlight](https://github.com/dsaltares/finlight-electron) is open source under the MIT license. Contributions are welcome! The stack I used is:

- Typescript
- React
- Electron
- React Router
- tRPC + React Query
- Kysely + SQLite
- Material UI
- recharts
